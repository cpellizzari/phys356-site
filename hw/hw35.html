HW 35 — Shallow Neural Network

📘 Related: Lesson 35, Lesson 36 🛠 MATLAB required 📁 Dataset: sat_data.mat

📋 Overview

In this assignment you will implement a complete shallow (one hidden layer) neural network from scratch in MATLAB — including Xavier initialization, the forward pass, backpropagation, and mini-batch SGD training. You will apply it to the sat_data.mat satellite maneuver classification dataset.

No toolboxes: You may not use MATLAB's Deep Learning Toolbox for this assignment. Implement every layer, activation, and gradient computation yourself. This builds the deep understanding that makes using toolboxes meaningful later.

📁 Dataset: sat_data.mat

The sat_data.mat file contains satellite sensor data. Load it with: 

load('sat_data.mat');
% Variables provided:
%   X_train  —  N_train × d  feature matrix (already normalized)
%   y_train  —  N_train × 1  binary labels (0 = nominal, 1 = maneuvering)
%   X_test   —  N_test  × d  feature matrix
%   y_test   —  N_test  × 1  binary labels

Features include radar cross-section over time, Doppler shift measurements, and derived orbital element residuals. Labels indicate whether the satellite performed an unscheduled maneuver in the observation window.

If sat_data.mat is not available: Generate a stand-in dataset using the code below. Your instructor will confirm which to use. 
% Synthetic stand-in (use only if sat_data.mat unavailable)
rng(356);
d = 12;  N_train = 800;  N_test = 200;
X_train = randn(N_train, d);
y_train = double(sum(X_train(:,1:3), 2) + randn(N_train,1) > 0);
X_test  = randn(N_test,  d);
y_test  = double(sum(X_test(:,1:3),  2) + randn(N_test,1)  > 0);

Problem 1 — Network Initialization

  1. (5 pts) Determine \(d\) from the loaded data. Choose an architecture: \(d \to n_1 \to 1\) with \(n_1 = 32\) hidden neurons. Report the total number of trainable parameters.
  2. (10 pts) Initialize parameters using Xavier initialization: \[\mathbf{W}^{[1]} \sim \mathcal{N}\!\left(0, \sqrt{\tfrac{2}{d}}\right), \quad \mathbf{W}^{[2]} \sim \mathcal{N}\!\left(0, \sqrt{\tfrac{2}{n_1}}\right)\] Initialize biases \(\mathbf{b}^{[1]} = \mathbf{0}\), \(b^{[2]} = 0\). Verify that the initial output of W2 * relu(W1 * X_train(1,:)' + b1) + b2 is in a reasonable range (i.e., not exploding). Report the mean and std of all initial pre-activations \(\mathbf{z}^{[1]}\) over the training set.

Problem 2 — Forward Pass

  1. (10 pts) Implement the vectorized forward pass for the full training set. Your implementation must use matrix operations (no loops over samples). Store \(\mathbf{Z}^{[1]}\), \(\mathbf{A}^{[1]}\), \(\mathbf{Z}^{[2]}\), \(\hat{\mathbf{p}}\) — you will need them in backprop. \[ \mathbf{Z}^{[1]} = \mathbf{W}^{[1]}\mathbf{X}^\top + \mathbf{b}^{[1]}, \quad \mathbf{A}^{[1]} = \text{ReLU}(\mathbf{Z}^{[1]}), \quad \mathbf{Z}^{[2]} = \mathbf{W}^{[2]}\mathbf{A}^{[1]} + b^{[2]}, \quad \hat{\mathbf{p}} = \sigma(\mathbf{Z}^{[2]})^\top \]
  2. (5 pts) Compute and report the initial cross-entropy loss on the training set.
  3. (5 pts) Compute and report the initial accuracy on the training set. Is it close to 50%? Should it be?

Problem 3 — Backpropagation

  1. (20 pts) Implement the backward pass. For each mini-batch, compute all gradients using the equations from Lesson 37: \[ \boldsymbol{\delta}^{[2]} = \hat{\mathbf{p}} - \mathbf{y}, \quad \frac{\partial L}{\partial \mathbf{W}^{[2]}} = \frac{1}{B}\boldsymbol{\delta}^{[2]\top} \mathbf{A}^{[1]\top}, \quad \frac{\partial L}{\partial b^{[2]}} = \frac{1}{B}\sum \delta^{[2]} \] \[ \boldsymbol{\delta}^{[1]} = (\mathbf{W}^{[2]})^\top \boldsymbol{\delta}^{[2]\top} \odot \mathbf{1}[\mathbf{Z}^{[1]} > 0], \quad \frac{\partial L}{\partial \mathbf{W}^{[1]}} = \frac{1}{B}\boldsymbol{\delta}^{[1]} \mathbf{X}_B^\top, \quad \frac{\partial L}{\partial \mathbf{b}^{[1]}} = \frac{1}{B}\boldsymbol{\delta}^{[1]}\mathbf{1} \]
  2. (10 pts) Verify your gradients with a numerical gradient check. For one scalar parameter \(w_{11}^{[1]}\), compute: \[\frac{\partial L}{\partial w_{11}^{[1]}} \approx \frac{L(w_{11}^{[1]}+\epsilon) - L(w_{11}^{[1]}-\epsilon)}{2\epsilon}, \quad \epsilon = 10^{-5}\] Report the analytic gradient, numerical approximation, and their relative error. Acceptable: relative error \(< 10^{-4}\).

Problem 4 — Mini-Batch SGD Training

  1. (15 pts) Implement a training loop with:
    • Mini-batch size \(B = 64\)
    • Learning rate \(\alpha = 0.01\)
    • 50 epochs
    • At each epoch: shuffle training data, iterate over mini-batches, compute loss and gradients, update all parameters
    • Record training loss per epoch
  2. (5 pts) Plot training loss vs. epoch. Is convergence smooth? Would you expect smoother convergence with larger or smaller batch size?
  3. (5 pts) Evaluate on the test set. Report test accuracy, test loss, precision, recall, and F1-score. Compare to the logistic regression results from HW 34 — did the neural network improve performance?

Problem 5 — Architecture Exploration (choose one)

Select one of the following extensions to explore:

  1. (10 pts) Hidden layer width: Train networks with \(n_1 \in \{4, 8, 16, 32, 64\}\). Plot test accuracy vs. \(n_1\). Is bigger always better?
  2. (10 pts) Learning rate schedule: Implement a simple cosine annealing schedule where \(\alpha(t) = \alpha_{\max}\cdot\frac{1+\cos(\pi t/T)}{2}\). Compare convergence to constant \(\alpha = 0.01\).
  3. (10 pts) Second hidden layer: Extend the network to \(d \to 32 \to 16 \to 1\). Implement the additional backprop equations and verify with a gradient check. Report test accuracy.
