HW 34 — Linear Classifier: CSpOC Space Objects

📘 Related: Lesson 34, Lesson 35 🛠 MATLAB required 📬 Submit: single .m or .mlx

📋 Overview

The Combined Space Operations Center (CSpOC) tracks approximately 27,000 objects in Earth orbit. A critical operational task is classifying radar returns as active satellites (y=1) vs. debris/rocket bodies (y=0). In this assignment you will build, train with gradient descent, and evaluate a logistic regression classifier on a simulated CSpOC-style dataset.

This assignment spans Lessons 34 and 35: Problem 1–2 cover the model setup (L34), Problem 3 covers gradient descent training (L35), and Problem 4 covers evaluation and analysis.

📖 Background

Features derived from radar observations include radar cross-section (RCS, a measure of reflectivity), visual magnitude, and orbital elements. Active satellites tend to have more controlled, predictable orbital behavior and different physical signatures than tumbling debris.

You will train a logistic regression model:

\[ \hat{p} = \sigma(\mathbf{w}^\top \mathbf{x} + b), \qquad \sigma(s) = \frac{1}{1+e^{-s}} \] \[ L = -\frac{1}{N}\sum_{i=1}^N \bigl[y_i \log\hat{p}_i + (1-y_i)\log(1-\hat{p}_i)\bigr] \]

And train it with gradient descent:

\[ \mathbf{w} \leftarrow \mathbf{w} - \alpha \cdot \frac{1}{N}\mathbf{X}^\top(\hat{\mathbf{p}} - \mathbf{y}), \qquad b \leftarrow b - \alpha \cdot \frac{1}{N}\mathbf{1}^\top(\hat{\mathbf{p}} - \mathbf{y}) \]

Problem 1 — Generate and Explore the Dataset (15 pts)

  1. (5 pts) Generate the dataset using the following code: 
    rng(356);
N  = 600;   % total samples

% Class 0: debris (low RCS, faint magnitude, high drag)
rcs0  = 0.5 + randn(N/2,1)*0.3;
mag0  = 5.5 + randn(N/2,1)*0.8;
drag0 = 1.2 + randn(N/2,1)*0.4;
X0    = [rcs0, mag0, drag0];

% Class 1: active satellites (higher RCS, brighter, lower drag)
rcs1  = 2.0 + randn(N/2,1)*0.4;
mag1  = 3.5 + randn(N/2,1)*0.7;
drag1 = 0.3 + randn(N/2,1)*0.2;
X1    = [rcs1, mag1, drag1];

X = [X0; X1];
y = [zeros(N/2,1); ones(N/2,1)];

% Shuffle
idx = randperm(N);
X   = X(idx,:);
y   = y(idx);
  2. (5 pts) Plot a 2D scatter plot of RCS vs. magnitude, color-coded by class. Do the classes appear linearly separable in this feature pair?
  3. (5 pts) Normalize the features (zero mean, unit variance, using training statistics). Split: 80% train, 20% test.

Problem 2 — Model Setup (Lesson 34) (20 pts)

  1. (5 pts) Initialize the logistic regression model: \(\mathbf{w} = \mathbf{0}\), \(b = 0\). Implement the sigmoid function as an anonymous function.
  2. (10 pts) Compute the initial forward pass on all training examples (vectorized, using matrix operations). Compute and report the initial cross-entropy loss and initial accuracy. What do you expect them to be for a random/zero model?
  3. (5 pts) Plot the initial decision boundary (derived from \(\mathbf{w}^\top\mathbf{x}+b=0\)) on top of the 2D scatter plot. Why is the boundary at the origin?

Problem 3 — Gradient Descent Training (Lesson 35) (35 pts)

  1. (20 pts) Implement the training loop using the update equations above. Use \(\alpha = 0.5\), 1000 iterations. Store the loss at each iteration.
    • At each iteration: forward pass → compute loss → compute gradients → update \(\mathbf{w}\) and \(b\).
    • Use fully vectorized MATLAB (no for loop over samples).
  2. (10 pts) Plot the loss curve (loss vs. iteration). Does it decrease monotonically? What is the final training loss?
  3. (5 pts) Repeat with \(\alpha = 0.01\) and \(\alpha = 2.0\). Show all three loss curves on one plot. Describe the behavior for each learning rate.

Problem 4 — Evaluation & Analysis (30 pts)

  1. (10 pts) Evaluate the trained model (\(\alpha=0.5\)) on the test set. Report:
    • Test accuracy (fraction of correctly classified examples)
    • Test cross-entropy loss
    • Confusion matrix (2×2 table of TP/TN/FP/FN)
    • Precision, Recall, and F1-score for class 1 (active satellites)
  2. (10 pts) Plot the learned decision boundary on the 2D scatter (RCS vs. magnitude) using the trained weights. Project the 3-feature boundary onto the 2D plane by setting drag to its mean value.
  3. (5 pts) Plot the distribution of predicted probabilities \(\hat{p}\) for each class (two overlapping histograms). How well-separated are the probability distributions?
  4. (5 pts) In space domain awareness, which error is more costly — a false positive (classifying debris as a satellite) or a false negative (classifying a satellite as debris)? Justify your answer. How would you adjust the classification threshold (default 0.5) to reduce your chosen error type?
