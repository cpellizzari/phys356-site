Backpropagation & Training Neural Networks

⏱ ~50 min read + in-class exercise

Learning Objectives

  • Apply the chain rule of calculus to compute gradients through composed functions.
  • Describe a computational graph and trace the forward and backward passes through it.
  • Derive the backpropagation equations for a two-layer network.
  • Explain the "closing the loop" relationship: backprop computes the gradients that SGD needs.
  • Implement a complete forward + backward pass for a shallow NN in MATLAB.

Notation used in this lesson

\(\delta^{[\ell]}\)Error signal at layer \(\ell\) (backprop term)
\(\frac{\partial L}{\partial \mathbf{W}^{[\ell]}}\)Gradient of loss w.r.t. layer weights
\(\phi'\)Derivative of activation function
\(\odot\)Element-wise (Hadamard) product

🌍 Background & Motivation

Closing the loop

In Lesson 35 we learned that gradient descent requires the gradient \(\nabla L\) to update parameters. In Lesson 36 we built a neural network with many layers, each with its own weights and biases. The question now is: how do we efficiently compute \(\partial L / \partial \mathbf{W}^{[\ell]}\) for every layer?

Backpropagation is the answer. It is an application of the chain rule from calculus, applied systematically to the computational graph of the neural network. It is not a learning algorithm itself — it is a gradient computation algorithm. SGD (Lesson 35) then uses those gradients to update parameters.

Historical note

Backpropagation was popularized in the 1986 paper by Rumelhart, Hinton, and Williams. The key insight was that gradients can be computed efficiently by a backward pass through the same graph used for the forward pass — reusing already-computed intermediate values.

🧠 Key Concepts

1. The Chain Rule (Review)

If \(L\) depends on \(z\), which depends on \(w\), then:

\[ \frac{dL}{dw} = \frac{dL}{dz} \cdot \frac{dz}{dw} \]

For a chain of functions \(L = f_n(f_{n-1}(\cdots f_1(w) \cdots))\):

\[ \frac{dL}{dw} = \frac{\partial f_n}{\partial f_{n-1}} \cdot \frac{\partial f_{n-1}}{\partial f_{n-2}} \cdots \frac{\partial f_1}{\partial w} \]

Backpropagation is simply this chain rule applied layer-by-layer from the output back to the input, accumulating gradient contributions.

2. Computational Graphs

A computational graph represents a mathematical expression as a directed acyclic graph (DAG). Each node is an operation; each edge carries a value and (in the backward pass) a gradient.

Example for a single neuron \(L = (\sigma(wx + b) - y)^2\):

w, x → [multiply] → wx → [add b] → z → [σ] → p → [subtract y] → e → [square] → L

The forward pass evaluates left-to-right, computing and storing intermediate values. The backward pass propagates \(\frac{\partial L}{\partial \text{each node}}\) right-to-left using the chain rule.

3. Backprop for a Two-Layer Network

Network: \(\mathbf{x} \to [\mathbf{W}^{[1]}, \mathbf{b}^{[1]}] \to \text{ReLU} \to [\mathbf{W}^{[2]}, b^{[2]}] \to \sigma \to L_{\text{BCE}}\)

Forward pass (recap):

\[ \mathbf{z}^{[1]} = \mathbf{W}^{[1]}\mathbf{x} + \mathbf{b}^{[1]}, \quad \mathbf{a}^{[1]} = \text{ReLU}(\mathbf{z}^{[1]}), \quad z^{[2]} = \mathbf{W}^{[2]}\mathbf{a}^{[1]} + b^{[2]}, \quad \hat{p} = \sigma(z^{[2]}) \]

Backward pass (backprop):

\[ \delta^{[2]} = \frac{\partial L}{\partial z^{[2]}} = \hat{p} - y \quad \text{(cross-entropy + sigmoid — they simplify beautifully!)} \] \[ \frac{\partial L}{\partial \mathbf{W}^{[2]}} = \delta^{[2]} \,(\mathbf{a}^{[1]})^\top, \qquad \frac{\partial L}{\partial b^{[2]}} = \delta^{[2]} \] \[ \delta^{[1]} = \frac{\partial L}{\partial \mathbf{z}^{[1]}} = (\mathbf{W}^{[2]})^\top \delta^{[2]} \odot \mathbf{1}[\mathbf{z}^{[1]} > 0] \quad \text{(chain rule through ReLU)} \] \[ \frac{\partial L}{\partial \mathbf{W}^{[1]}} = \delta^{[1]} \mathbf{x}^\top, \qquad \frac{\partial L}{\partial \mathbf{b}^{[1]}} = \delta^{[1]} \]
Key pattern: Each layer's gradient depends on the error signal from above (\(\delta^{[\ell+1]}\)) multiplied back through the weight matrix and gated by the activation derivative. This is the "backpropagated error" interpretation.

4. The ReLU Derivative

\(\text{ReLU}'(z) = \mathbf{1}[z > 0]\) — it is 1 where the neuron was active, 0 where it was inactive. This is why the expression \(\odot \mathbf{1}[\mathbf{z}^{[1]} > 0]\) appears: we "gate" the gradient through neurons that fired.

Dying ReLU: If a neuron's pre-activation \(z\) is always negative, its gradient is always 0 — it never learns. This is called a "dead neuron." Careful initialization and moderate learning rates prevent this.

5. The General Backprop Algorithm

For a network with \(L\) layers and mini-batch of \(B\) examples:

  1. Forward pass: Compute and store \(\mathbf{z}^{[\ell]}\), \(\mathbf{a}^{[\ell]}\) for all \(\ell = 1, \ldots, L\).
  2. Compute output gradient: \(\delta^{[L]} = \nabla_{z^{[L]}} L\).
  3. Backpropagate from \(\ell = L-1\) down to \(1\): \(\delta^{[\ell]} = (\mathbf{W}^{[\ell+1]})^\top \delta^{[\ell+1]} \odot \phi'(\mathbf{z}^{[\ell]})\).
  4. Compute weight gradients: \(\frac{\partial L}{\partial \mathbf{W}^{[\ell]}} = \frac{1}{B}\delta^{[\ell]} (\mathbf{A}^{[\ell-1]})^\top\).
  5. Update parameters with SGD: \(\mathbf{W}^{[\ell]} \leftarrow \mathbf{W}^{[\ell]} - \alpha \frac{\partial L}{\partial \mathbf{W}^{[\ell]}}\).

💻 Worked Example — Full Forward + Backward Pass

This implements both the forward and backward pass for a 1-hidden-layer network. Compare this directly to the forward-only code in Lesson 36: 

%% Lesson 37 — Full Backpropagation Example
% Network: d → n1 → 1  (binary classification)
% Activation: ReLU (hidden), Sigmoid (output)

rng(42);
d  = 4;   n1 = 8;   % small network for clarity
N  = 50;            % batch size

% Synthetic data
X = randn(d, N);                % d × N
y = double(sum(X) > 0)';        % N × 1 labels

% Initialize parameters (Xavier)
W1 = randn(n1,d)  * sqrt(2/d);  b1 = zeros(n1,1);
W2 = randn(1, n1) * sqrt(2/n1); b2 = 0;

relu    = @(z) max(0, z);
sigmoid = @(z) 1 ./ (1 + exp(-z));
alpha   = 0.05;  eps = 1e-8;

loss_hist = zeros(200,1);

for t = 1:200
    %% ===== FORWARD PASS =====
    Z1 = W1 * X + b1;          % n1 × N
    A1 = relu(Z1);              % n1 × N

    Z2 = W2 * A1 + b2;         % 1 × N
    P  = sigmoid(Z2)';          % N × 1  — predicted probabilities

    % Binary cross-entropy loss
    L_val = -mean(y.*log(P+eps) + (1-y).*log(1-P+eps));
    loss_hist(t) = L_val;

    %% ===== BACKWARD PASS =====
    % Output layer error
    dZ2 = (P - y)';             % 1 × N  (chain rule: dL/dZ2 = p_hat - y)

    % Gradients for W2, b2
    dW2 = (dZ2 * A1') / N;     % 1 × n1
    db2 = sum(dZ2, 2) / N;     % scalar

    % Backprop through hidden layer
    dA1 = W2' * dZ2;            % n1 × N
    dZ1 = dA1 .* double(Z1>0); % n1 × N  (ReLU gate)

    % Gradients for W1, b1
    dW1 = (dZ1 * X') / N;      % n1 × d
    db1 = sum(dZ1, 2) / N;     % n1 × 1

    %% ===== SGD UPDATE =====
    W2 = W2 - alpha * dW2;
    b2 = b2 - alpha * db2;
    W1 = W1 - alpha * dW1;
    b1 = b1 - alpha * db1;
end

%% Results
fprintf('Final loss: %.4f\n', loss_hist(end));
P_final = sigmoid(W2 * relu(W1*X + b1) + b2)';
acc = mean(double(P_final >= 0.5) == y);
fprintf('Training accuracy: %.2f%%\n', acc*100);

figure;
plot(loss_hist, 'b-', 'LineWidth', 2);
xlabel('Iteration'); ylabel('Cross-Entropy Loss');
title('Backprop + SGD Training'); grid on;
Key line to study: dZ1 = dA1 .* double(Z1>0) — this is the chain rule applied to ReLU. double(Z1>0) is the derivative: 1 where the neuron fired, 0 where it did not.

✏️ In-Class Exercise (Printable)

Work through this by hand. Show all steps.

Setup

Consider a tiny network with \(d=2\), \(n_1=2\), output \(n_2=1\).

Given (at a particular training step):

\[ \mathbf{x} = \begin{bmatrix}1\\2\end{bmatrix}, \quad y = 1, \quad \mathbf{W}^{[1]} = \begin{bmatrix}0.5 & -0.3\\0.1 & 0.8\end{bmatrix}, \quad \mathbf{b}^{[1]} = \begin{bmatrix}0\\0\end{bmatrix} \] \[ \mathbf{W}^{[2]} = \begin{bmatrix}0.4 & -0.6\end{bmatrix}, \quad b^{[2]} = 0.1 \]

Tasks

  1. (a) Compute \(\mathbf{z}^{[1]}\) and \(\mathbf{a}^{[1]} = \text{ReLU}(\mathbf{z}^{[1]})\).
  2. (b) Compute \(z^{[2]}\) and \(\hat{p} = \sigma(z^{[2]})\).
  3. (c) Compute the binary cross-entropy loss \(L\) for this single example.
  4. (d) Backprop: compute \(\delta^{[2]} = \hat{p} - y\).
  5. (e) Compute \(\frac{\partial L}{\partial \mathbf{W}^{[2]}}\) and \(\frac{\partial L}{\partial b^{[2]}}\).
  6. (f) Backprop through layer 1: compute \(\delta^{[1]} = (\mathbf{W}^{[2]})^\top \delta^{[2]} \odot \mathbf{1}[\mathbf{z}^{[1]}>0]\).
  7. (g) Compute \(\frac{\partial L}{\partial \mathbf{W}^{[1]}}\) and \(\frac{\partial L}{\partial \mathbf{b}^{[1]}}\).
  8. (h) With \(\alpha = 0.1\), perform one SGD update on all parameters. What are the new values of \(\mathbf{W}^{[1]}\)?

Expected Answers (Check your work)

Click to reveal

(a) \(\mathbf{z}^{[1]} = [-0.1,\ 1.7]^\top\), \(\mathbf{a}^{[1]} = [0,\ 1.7]^\top\) (ReLU zeros the negative)

(b) \(z^{[2]} = 0.4(0) + (-0.6)(1.7) + 0.1 = -0.92\), \(\hat{p} = \sigma(-0.92) \approx 0.285\)

(c) \(L = -\log(0.285) \approx 1.255\)

(d) \(\delta^{[2]} = 0.285 - 1 = -0.715\)

(e) \(\frac{\partial L}{\partial \mathbf{W}^{[2]}} = \delta^{[2]}\mathbf{a}^{[1]\top} = [-0.715 \cdot 0,\ -0.715 \cdot 1.7] = [0,\ -1.215]\); \(\frac{\partial L}{\partial b^{[2]}} = -0.715\)

(f) \((\mathbf{W}^{[2]})^\top \delta^{[2]} = [-0.286,\ 0.429]^\top\); gate with \([0,\ 1]\): \(\delta^{[1]} = [0,\ 0.429]^\top\)

(g) \(\frac{\partial L}{\partial \mathbf{W}^{[1]}} = \delta^{[1]}\mathbf{x}^\top = \begin{bmatrix}0&0\\0.429&0.858\end{bmatrix}\)

📝 Summary

ConceptKey Idea
Chain rule\(\frac{dL}{dw} = \frac{dL}{dz}\cdot\frac{dz}{dw}\) — applied layer by layer backward
Output delta\(\delta^{[L]} = \hat{p} - y\) for BCE + sigmoid (elegant simplification)
Hidden delta\(\delta^{[\ell]} = (\mathbf{W}^{[\ell+1]})^\top\delta^{[\ell+1]} \odot \phi'(\mathbf{z}^{[\ell]})\)
Weight gradient\(\frac{\partial L}{\partial \mathbf{W}^{[\ell]}} = \delta^{[\ell]}(\mathbf{a}^{[\ell-1]})^\top\)
Full training loopForward pass → loss → backward pass → SGD update → repeat
